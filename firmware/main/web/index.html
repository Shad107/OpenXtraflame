Options avancées

BETA

Cloud Extraflame (=TotalControl 2)

Active la connexion sortante vers mqtt.extraflame.it:8883 en parallèle du bridge HA local. Permet à l'app TotalControl 2 officielle de continuer à voir le poêle.
Nécessite un compte TotalControl 2 (=même credentials que l'app mobile) car le firmware récupère l'ID interne du poêle via REST appapi.extraflame.it/auth.

RECOVERY

Safe mode (=prochain boot)

Force le prochain reboot en mode minimal : WiFi + web UI + OTA seulement. Aucun Micronova, MQTT, ni cloud. À utiliser si un firmware buggué crashe en boucle et empêche l'OTA de passer. Le flag est consommé au boot (=un 2e reboot ramène en normal).

EXPERIMENTAL

Guardian Mode

⚠️ Fonctionnalité avancée. Guardian Mode connecte votre module au cloud Extraflame en écoute uniquement pour capturer les mises à jour officielles vers votre serveur archive personnel. Aucune donnée utilisateur transmise. En savoir plus
